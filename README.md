## 🚀 Featured Projects

### [**ShelterLab**](https://github.com/keirara04/ShelterLab) — Campus Marketplace Platform
**🎓 University-Exclusive Peer-to-Peer Marketplace**

A full-stack web application enabling Korean university students to buy and sell used items within a trusted community.

**Key Achievements:**
- 🔐 **Trust System:** LabCred credibility scoring (4-tier progression, transparent algorithm)
- 🏫 **Multi-University Support:** 10+ Korean campuses with campus-exclusive feeds
- 🤖 **AI Integration:** Groq Mixtral LLM for intelligent pricing suggestions
- 📊 **Admin Dashboard:** Real-time statistics, user approval workflow, moderation tools
- 🔒 **Security:** Rate limiting (Upstash Redis), JWT auth, RLS policies, PIPA compliance
- 📱 **PWA:** Native app installation, offline support, push notifications
- 📧 **Email Service:** OTP verification via Brevo SMTP

**Tech Stack:** Next.js 16 • React 19 • Tailwind CSS • Supabase • Groq API • Upstash Redis • Web Push  
**Status:** 0.1.2-beta | [Live Demo](https://shelterlab.shop)
