## Featured Project

### [ShelterLab](https://github.com/keirara04/ShelterLab), Campus Marketplace Platform

A full-stack peer-to-peer marketplace for Korean university students, supporting 10+ campuses with campus-exclusive feeds.

**What it does:**
- LabCred trust system, a transparent, tiered credibility score for marketplace users
- AI-assisted pricing suggestions using the Groq LLM API
- Admin dashboard with moderation tools and live usage statistics
- Rate limiting (Upstash Redis), JWT auth, and Postgres row-level security policies, built with PIPA compliance in mind
- Progressive Web App: offline support, installable, push notifications
- Email verification via Brevo SMTP

**Stack:** Next.js, React, Tailwind CSS, Supabase, Groq API, Upstash Redis

**Status:** 0.1.2-beta | [Live demo](https://shelterlab.shop)
